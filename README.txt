DOMAIN EXPANSION — scroll experience
====================================

HOW TO RUN
  Keep all files in one folder and open index.html in a browser.


REPLACE THESE FILES (keep the exact names)
  hero.mp4   Hero background video. Looping, dark & moody works best.
             Included file is a temporary dark-gradient loop.
  1.png      LEFT hand  — transparent PNG, hand reaching toward the right.  (real image included)
  2.png      RIGHT hand — transparent PNG, hand reaching toward the left.   (real image included)
  3.png      FORMED HAND SIGN — transparent PNG, wide (~1500x550).
             Shown huge at screen center while the storm rages around it.
  4.png      CHARACTER FULL BODY — transparent or dark-background PNG,
             tall (~900x1600). Rises from the bottom center in the finale.
  5.png      TEMPLE / DOMAIN BACKGROUND — transparent PNG, wide (~1500x950).
             Rises behind the character (slower = parallax depth).
  6.png      TITLE ART ("DOMAIN EXPANSION") — transparent PNG (~680x350).
             Slides into the bottom-left of the final scene.

NOTES
  - All lightning is generated in code (canvas engine in index.html):
    fractal channels, white-hot core, violet glow, restrike flicker,
    ambient distant strikes + sheet lightning. No lightning images needed.
  - Scroll flow: HERO (video + kanji) -> hands meet -> sign + full-screen
    thunder -> temple + character parallax reveal.
  - Kanji text in the hero can be edited directly in index.html
    (search for "kanjiRail" and "kanjiCorner").
